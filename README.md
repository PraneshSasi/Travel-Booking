# SkyNest Travel Booking

A premium travel booking landing page featuring a modern editorial design, warm organic color palette, seamless animated layout, and 3D floating components.

## Features
- **Editorial Aesthetic:** Warm terracotta, sage grey, and champagne frosted glass colors paired with elegant Cormorant Garamond typography.
- **Dynamic Search & Booking:** Interactive booking widget matching core search functionalities (Hotels, Things to Do, Restaurants, Flights, Holiday Homes).
- **Split-Pane Authentication:** Distinctively designed modal for Login and Sign Up with smooth transitions and an interactive password visibility toggle.
- **Floating 3D Animations:** GPU-accelerated smooth floating animations for visual imagery.
- **Fluid Golden Wave Background:** A dynamic 3D particle grid using Three.js that responds smoothly to scrolling and mouse movement.

## Tech Stack
- HTML5
- Vanilla CSS (Custom properties, Keyframe Animations, Grid/Flexbox)
- Vanilla JavaScript (DOM Manipulation, Interactive state management)
- [Three.js](https://threejs.org/) (for 3D particle fluid background)

## Local Setup
1. Clone the repository.
2. Open `index.html` in your web browser. No build steps or node modules required!

## Deployment
All static files (`index.html`, `style.css`, `script.js`) can be hosted directly on any static web server like GitHub Pages, Vercel, or Netlify.
